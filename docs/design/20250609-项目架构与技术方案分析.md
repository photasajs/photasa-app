## 一、项目概览

Photasa 是一款基于 Electron + Vue3 的桌面照片管理应用，支持多平台，具备高性能缩略图生成、文件扫描、元数据管理、配置持久化等能力。采用主进程-预加载-渲染进程三层架构，充分利用 worker 线程实现异步与高并发处理。

---

## 二、核心技术栈

- Electron（桌面应用壳，主进程/预加载/渲染进程分离）
- Vue3 + Pinia（前端 UI 与状态管理）
- Ant Design Vue（UI 组件库）
- Vite（前端构建工具）
- TypeScript（全栈类型安全）
- sharp、heic-decode、fluent-ffmpeg（图片/视频缩略图处理）
- fs-extra、klaw、chokidar（文件系统操作与监听）
- log4js、pino（日志体系）
- rxjs、vue-concurrency（异步与任务调度）
- ffmpeg-static、ffprobe-static（二进制依赖管理）
- 其他：ramda、moment、concurrent-tasks 等

## 三、架构分层与主要模块

### 1. 主进程（src/main）

- 负责窗口管理、IPC 通信、worker 线程调度、文件系统操作、配置与缩略图服务。
- 主要模块：
    - index.ts：主入口，初始化服务与窗口
    - config-service/handler/storage/worker：配置系统全链路
    - scan-photos/scan-service/scan-worker：文件扫描与元数据管理
    - thumbnail-handler/service/worker：缩略图生成与缓存
    - fs-watch：文件变更监听

### 2. 预加载层（src/preload）

- 通过 contextBridge 安全暴露 API 给渲染进程
- 主要功能：文件操作、EXIF 解析、路径处理、图片导入、shell 调用等

### 3. 渲染进程（src/renderer/src）

- Vue3 SPA，负责 UI、用户交互、状态管理
- 主要模块：
    - App.vue：主入口，调度扫描与 UI
    - components/：图片列表、文件夹树、设置、导入等 UI 组件
    - stores/：Pinia 状态（preference、photos）
    - utils/：API 封装、文件/图片/任务处理
    - i18n/locales：多语言支持
    - worker/：前端 worker 工具

### 4. 公共模块（src/common）

- 类型定义、通用工具、日志、批处理、错误处理等

## 四、核心业务流程

### 1. 文件夹选择与偏好存储

- 用户通过 UI 选择文件夹，路径存入 Pinia preference store，并持久化到 localStorage

### 2. 文件扫描与数据库

- 主进程/worker 递归扫描文件夹，收集文件元数据，存入 Pinia photos store
- 支持批量、异步、并发处理，优化大规模目录性能

### 3. 缩略图生成

- 针对图片/视频/HEIC 文件，分别用 sharp、heic-decode、ffmpeg 生成缩略图
- 采用 worker 线程异步处理，结果缓存于 photasaoriginals 目录

### 4. 配置系统

- 多进程协作，.photasa.json 文件存储每个目录/文件的配置
- 支持缓存、批量、队列、优先级、超时等机制，提升性能与可靠性

### 5. UI 展示与交互

- 图片懒加载、缩略图快速预览、点击大图预览
- 文件夹树、批量导入、设置、主题切换、多语言
